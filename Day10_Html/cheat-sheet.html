🚀 HTML Cheatsheet for Developers

Page Setup & Metadata

<!DOCTYPE html> → Always include (HTML5)

<meta charset="UTF-8"> → Unicode support

<meta name="viewport" content="width=device-width, initial-scale=1.0"> → Mobile friendly

<title> → Browser tab title

<link rel="icon" href="favicon.png"> → Adds favicon

Essential Tags

<header> → Top section (logo, nav)

<main> → Core page content

<footer> → Bottom credits/info

<nav> → Navigation menus

<section> / <article> → Logical content blocks

Links & Buttons

<a href="page.html">Visit</a>

target="_blank" → Opens in new tab

download → Forces download

rel="noopener noreferrer" → Security for new tab

<button> → For actions (JS/submit)

Images & Media

<img src="img.jpg" alt="desc"> → Must include alt

loading="lazy" → Improves performance

<video controls poster="thumb.jpg">

<audio controls>

<source> → Multiple formats support

Forms (Modern)

<form action="/submit" method="post">

<input type="email" required>

<input type="password" minlength="6">

<input type="file" accept="image/*">

<button type="submit">Send</button>

SEO & Accessibility

<meta name="description" content="Site info">

alt → Always describe images

<label for="id"> → Ties label to input

aria-label → Describes for screen readers

<h1> → One per page, main title

Useful Attributes

id → Unique element name

class → Grouping for CSS/JS

data-* → Custom data for JS

title → Tooltip on hover

hidden → Hides element (still in DOM)

Layout & Structure

<div> → Container (block)

<span> → Inline text wrapper

<br> / <hr> → Breaks/lines

<figure> + <figcaption>

<aside> → Side info or ads

Performance Tips

✅ Use loading="lazy" for images

✅ Minimize inline CSS & JS

✅ Use semantic tags (better SEO)

✅ Compress images before upload

✅ Place scripts before </body>

Fav Combos (HTML + Dev)

✅ HTML + CSS → Styling

✅ HTML + JS → Interactivity

✅ HTML + SEO Meta → Ranking

✅ HTML + ARIA → Accessibility

✅ HTML + CDN → Faster loading
